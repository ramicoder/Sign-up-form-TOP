# The Odin Project - Sign-Up Form

## Overview
This project is part of the Odin Project's Intermediate HTML & CSS course.  
It is a **sign-up form** for an imaginary service, designed to practice:

- HTML structure and form elements
- CSS layout using flexbox
- Styling inputs, buttons, and pseudo-classes (`:focus`, `:invalid`)
- Adding background images and semi-transparent overlays
- Using custom fonts
- Linking external assets and giving image credits

## Features
- Left-side screen with a background image and logo
- Semi-transparent logo container for readability
- Right-side screen with text content and a sign-up form
- Form fields: First Name, Last Name, Email, Phone Number, Password, Confirmation Password
- Styled inputs with focus and invalid states
- Submit button with a custom style
- Image credit and proper attribution for external assets

- Open `index.html` in a web browser
- Fill in the form fields to see the styling and validation effects
